About On Time


What is On Time?

Metro weekly is a user contributed weekly news letter related to Windows® 8, Windows® Phone, and/or Microsoft™ XBox and related technologies. Users such as yourself submit news worthy articles and cool things related to the aforementioned technologies and we at Metro-Weekly

A portable version of the application (published under the same title) for your Windows® Phone can be downloaded from the Windows® Phone market place."

Why did we do this?

The website was created as a tool to share interesting information with fellow metro heads. As Windows 8 was released to the developers, Clark Sell and Lwin Maung decided that the website can actually have a face in the Windows Store since features of Windows 8 can help users such as yourself makes sharing / submitting of interesting articles easier.

How was this created?

The pre-compiled version of the application can be downloaded from GitHub at : http://www.github.com/csell5/metro-weekly/

Who made this application?

Website is a creation of Clark Sell. The Windows 8 application is a creation between Clark Sell and Lwin Maung and code contributions from users around the web.
